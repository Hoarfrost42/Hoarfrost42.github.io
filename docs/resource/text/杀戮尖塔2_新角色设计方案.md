# 杀戮尖塔2 新角色设计方案

## 设计方案来源
- 这是作为给自己练习理解拆解游戏思路的试手，还未实际实装为Mod。偏向于游戏内其他角色的对抗体验+脑测。


## 角色概述

**角色身份：** 外星过客 / 混乱散播者——一个偏反派定位的DLC角色，与尖塔世界观中的其他角色不同，它只是个路过时顺手播撒混乱的存在。

**设计思路：** 现有角色都在"敌人攻击我，我如何应对"的框架内做文章。这个角色的核心逻辑是——"我不让你按计划行动"。通过改变敌人意图来获取策略空间，而不是单纯靠数值碾压。不与敌人交互的操作本身也是一种交互。

**商业化定位：** 作为DLC付费角色发售，可配套独立的游戏路线或剧情线，以差异化内容吸引玩家消费。

**与现有角色差异：** 其他角色的核心循环围绕"攻击-防御-强化"展开，本角色的核心循环围绕"控制-积蓄-爆发"展开。联机模式下天然承担控制位，通过改变敌人意图减轻队友压力。

## 核心机制：迷茫（Bewilderment）

### 基础规则

"迷茫"是本角色的专属Debuff，代表敌人被混乱力量侵蚀后逐渐失去行动方向。

**机制链路：** 叠加迷茫 → 达到阈值触发眩晕（改变意图为不行动）→ 眩晕期间继续叠层 → 第二阈值触发爆发伤害 → 眩晕结束，循环重置。

### 关键规则细节（待定数值用方括号标注）

- 迷茫本身是一个可被人工制品（Artifact）阻挡的Debuff。
- 迷茫叠加至 [10] 层时，敌人进入"临界"状态。临界状态下，迷茫不再自动触发眩晕，而是需要玩家主动打出一张特定技能牌来引爆。
- 设计"临界→手动引爆"而非自动触发的原因：增加决策点。玩家需要判断当前回合敌人意图是否值得打断——如果敌人下回合本来就是弱行动，保留临界状态可能更有价值（例如这本身就是BOSS给自己上BUFF、给玩家上DEBUFF的某一环节）。
- 眩晕后迷茫层数自动清零。眩晕期间继续叠加迷茫，每叠至 [10] 层时可触发一次爆发伤害，造成 [15] 点固定伤害，同时迷茫层数清零。
- 眩晕持续一回合后自然解除。解除后迷茫层数清零，循环重置。
- 不设置硬性的"隔回合才能再眩晕"限制——通过临界→手动引爆的机制，玩家自然会因为策略考量而非强制冷却来控制节奏。

### 与人工制品的关系

迷茫作为Debuff会被人工制品阻挡，但"改变意图"这一效果本身不在人工制品的克制范围内。这意味着角色有两条路径突破人工制品：一是用常规Debuff（如虚弱、易伤）先消耗人工制品层数再叠迷茫；二是通过高稀有度卡牌直接改变意图，绕过迷茫叠层流程。可以考虑机制/遗物/卡牌效果三个方向来继续优化，给予一个效果：若敌人被眩晕，则去除敌方所有的人工制品层数。

## 被动机制（角色固有）



设计意图：将防御与核心机制绑定，而非依赖独立的防御卡体系。前期叠层手段少时防御薄弱，中后期卡组成型后防御随叠层效率自然提升，形成"前期脆弱、后期稳健"的角色功率曲线。

### 诅咒牌联动

作为混乱散播者，诅咒对本角色而言不是纯粹的负面——但也不应该强到让玩家主动追求诅咒牌。

设计思路：设置一张特定的能力牌或技能牌，可以将手中的诅咒牌"投射"给敌人，转化为等量的迷茫层数。这让诅咒从"死牌"变成"可用的资源"，但不改变诅咒牌占手牌位的基础惩罚。玩家的体验是诅咒没有那么卡手，也没有那么纯粹的低收益，毕竟2代没有了1代的黑石项链和巫毒娃娃这类和诅咒形成联动的遗物。

## 场景适配分析

### 普通战斗

初始卡组的"混沌打击"提供基础叠层+伤害，配合被动机制的格挡，前几场战斗可以正常推进。第一幕的核心目标是获取至少一张高效叠层卡（如"深层侵蚀"）和引爆牌。

### 精英战

精英战通常是单体高血量敌人，正是本角色最擅长的场景。完整的"叠层→临界→引爆→眩晕期间继续叠层→爆发伤害"循环可以在单体上打出很高的总伤害。

### 群体战斗

依赖"蔓延之触"（AOE叠层）和"混沌传染"（层数复制）解决多目标问题。群体场景的策略是先集中叠层一个目标，再通过传染扩散。初始遗物"混沌种子"的全体2层迷茫也提供了群体场景的起步支持。

### Boss战

Boss战是最需要决策深度的场景。很多Boss有固定的意图循环，"临界→手动引爆"的机制让玩家需要判断最佳打断时机——在Boss准备放大招时引爆价值最高，在Boss做弱行动时引爆则浪费了控制资源。这种"读意图→选择时机"的博弈是本角色在Boss战中的核心乐趣。

需要注意的风险：改变意图后Boss可能roll出更强的行动。这是双刃剑的设计，确保控制不是无脑有利的。

### 联机模式

控制型定位在联机中天然有价值。队友可以在敌人被眩晕的回合集中输出，或者利用玩家提供的安全窗口进行自身强化。"恐慌扩散"等抽牌型能力牌也保证了联机中的卡组运转效率。

- 各项数值（迷茫阈值、爆发伤害倍率、卡牌具体数值）需要通过实际测试和数值模拟来确定平衡性。

- "混乱"（让敌人攻击敌人）作为稀有卡的具体实现方式和规则定义。
- 角色专属事件和剧情线的设计（配合DLC商业化定位）。
- 各幕典型卡组构建路线（Build Path）的梳理。
